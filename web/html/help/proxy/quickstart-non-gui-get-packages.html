RHN Proxy Server 2.1: Installation Guide
Appendix F. Quick Start Guide for Manual Installation and Configuration

F.2. Install Proxy Packages

Use the following steps to install all the packages for RHN Proxy Server on the target machine:

  1. Using either the Red Hat Update Agent or the Spacewalk Registration Client, register the system with Spacewalk.

  2. Log in to the RHN website at https://rhn.redhat.com. Click Systems in the top navigation bar and then the name of the system in the system list. From the System Details page, click the Properties subtab, select the Enterprise entitlement, and click the Update Properties button.

  3. While in the System Details page, click the Channels tab and select the channel associated with the server's base operating system, Red Hat Enterprise Linux AS 2.1. Then click the Change Subscriptions button.

  4. In the System Details page, click the RHN Proxy subtab and activate the system's RHN Proxy Server license.

  5. From a shell prompt on the target system, execute the command up2date up2date to ensure the latest version of the Red Hat Update Agent is installed. If prompted to install the Red Hat GPG key, do so and then run up2date up2date again.

  6. Execute the command up2date -u on the target system to ensure that the latest versions of all packages are installed. This process may take up to 15 or 20 minutes.

  7. Execute the command up2date rhns-proxy to download all RHN Proxy Server packages.

  8. Execute the command up2date -u once again to make sure all packages are updated.
