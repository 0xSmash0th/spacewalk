RHN Proxy Server 2.1: Installation Guide

Appendix F. Quick Start Guide for Manual Installation and Configuration

This chapter contains a Quick Start Guide to help you get a basic RHN Proxy Server working using the manual configuration process (non-GUI). It is strongly recommended you use the graphical installation program discussed in Chapter 4 Installation. Only expert users should attempt a manual installation and configuration of the RHN Proxy Server. Many potential problems may arise, especially in regard to security. If you choose note to run X on the system, follow these instructions instead.

F.1. Start With a Fresh Install

Install a clean copy of Red Hat Enterprise Linux AS on the target system. RHN Proxy Server is supported only on Red Hat Enterprise Linux AS 2.1. When installing the operating system on the target system, select the server-class installation method and the Web server packages. Create at least one user account other than the root account. Set the system clock to UTC.

Warning
 

There is great time sensitivity when connecting to a Web server running SSL; it is imperative the time settings on the clients and server are reasonably close together so the SSL certificate does not expire too quickly. It is recommended NTP be used to synchronize the clocks.
