RHN Proxy Server 2.1: Installation Guide

Appendix C. RHN Proxy Server Configuration File Variables

This appendix contains the default values for the variables in the /etc/rhn/rhn.conf configuration file and whether or not they are required variables. If the variable is listed as not required, the default value is acceptable.

traceback_mail

Default Value:

Required: Yes

debug

Default Value: 5

Required: No

quiet_mail

Default Value: 15

Required: No

proxy.broker.traceback_mail

Default Value:

Required: No

proxy.broker.debug

Default Value: 5

Required: No

proxy.broker.quiet_mail

Default Value: 15

Required: No

proxy.broker.rhn_parent

Default Value: redirect_machine.mycompany.com/XMLRPC_REDIRECT

Required: Yes, if requiring a proper secure connect via an RHN Proxy SSL Redirect Server (strongly encouraged)

proxy.broker.http_proxy

Default Value: 127.0.0.1:8080

Required: No, unless the HTTP proxy caching server (Squid) resides on a different machine

proxy.broker.http_proxy_username

Default Value:

Required: No

proxy.broker.http_proxy_password

Default Value:

Required: No

proxy.broker.auth_cache_server

Default Value: 127.0.0.1

Required: No, unless the RHN Authentication Daemon resides on a different machine

proxy.broker.auth_cache_server_port

Default Value: 9999

Required: No, unless the RHN Authentication Daemon is configured to use a different port

proxy.broker.pkg_dir

Default Value: /var/up2date/packages

Required: No

proxy.redirect.traceback_mail

Default Value:

Required: No

proxy.redirect.debug

Default Value: 5

Required: No

proxy.redirect.rhn_parent

Default Value: xmlrpc.rhn.redhat.com/XMLRPC

Required: No

proxy.redirect.http_proxy

Default Value:

Required: No, unless the redirect must pass through a corporate http proxy gateway.

proxy.redirect.http_proxy_username

Default Value:

Required: No

proxy.redirect.http_proxy_password

Default Value:

Required: No

proxy.redirect.ca_chain

Default Value: /usr/share/rhn/RHNS-CA-CERT

Required: No

proxy.package_manager.rhn_parent

Default Value: xmlrpc.rhn.redhat.com/XP

Required: No

proxy.package_manager.http_proxy

Default Value:

Required: No, unless the package manager must pass through a corporate http proxy gateway.

proxy.package_manager.http_proxy_username

Default Value:

Required: No

proxy.package_manager.http_proxy_password

Default Value:

Required: No

proxy.package_manager.ca_chain

Default Value: /usr/share/rhn/RHNS-CA-CERT

Required: No

auth_cache.traceback_mail

Default Value:

Required: No

auth_cache.debug

Default Value: 5

Required: No

auth_cache.auth_cache_server_port

Default Value: 9999

Required: No

auth_cache.allowed_hosts

Default Value: 127.0.0.1, localhost

Required: No, unless the RHN Authentication Daemon is not local to the RHN Proxy Broker Server.
