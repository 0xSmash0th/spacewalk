Chapter 4. Red Hat Update Agent

Installing Updated Packages

After downloading the packages through the Red Hat Update Agent, they must be installed. If you chose not to have the packages installed via the Red Hat Update Agent, skip to the Section called Manual Package Installation for further instructions. If you configured it to installed the packages (the default setting), the selected packages are installed. The progress of installing each package, as well as the total progress, is displayed. When the packages have been installed, as seen in Figure 4-10, click Next to continue.
