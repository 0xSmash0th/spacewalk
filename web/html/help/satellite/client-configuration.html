Spacewalk Server: Installation and Configuration Guide; Version 0.3

Chapter 5. Client Configuration

This chapter describes the steps necessary to configure client systems to use Spacewalk Server.

5.1. Configuring Client SSL Certificates

Since the user will be connecting to a private Web server that will not have a publicly available certificate, it is necessary to install the certificate for the Spacewalk Server directly on each client.

An RPM to install this certificate is created during the Spacewalk Server installation and will be placed in the /pub directory on the Web server.

From each client, the user may download the certificate using a Web browser. Once downloaded, the user may install the certificate by running rpm -Uvh certificate-package-name.
