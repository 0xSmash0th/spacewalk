Appendix A. Configuring Oracle

A.2. Troubleshooting

The installation of the database schema is intended to be as painless as possible. Assuming that the installation procedure is followed as written, the schema should install with no errors. If, however, there are problems with the database itself, please contact the RHN team at <rhn-satellite@redhat.com>.
