Chapter 5. Maintenance

5.2. Upgrading the Spacewalk Application

If any critical updates are made to Spacewalk Server, they will be released in the form of an errata for the Spacewalk Server.

For Spacewalk Server systems that may be connected to the Internet, the best method for applying these Errata Updates is using the Red Hat Update Agent via Spacewalk. Since the Spacewalk Server is subscribed to Spacewalk during initial installation, the user should be able to run up2date -u on the Spacewalk Server or use the website at https://rhn.redhat.com to apply the updates.

For Spacewalk Server systems that may not be connected to the Internet, the packages themselves may be retrieved using a customer account at https://rhn.redhat.com. Then, they can be applied manually by the customer according to instructions in the Errata Advisory.

Warning
 

It is very important to read the Errata Advisory before applying any Spacewalk Server Errata Updates. Additional configuration steps may be required to apply certain Spacewalk Server updates, especially if they involve the Oracle database. In such cases, the advisory will contain specific and detailed information about necessary steps that may be required.
