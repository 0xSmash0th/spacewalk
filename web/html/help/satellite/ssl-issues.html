Chapter 5. Client Configuration

5.3. SSL Connection Issues

SSL connection to local servers is not supported in some older versions of the Spacewalk Registration Client and the Red Hat Update Agent. This is due to the inability of these older clients to understand server keys other than the central Spacewalk Server keys.

For client versions in which localized SSL is not supported, non-SSL values must be used in the configuration files.

Newer client packages have been released for all versions of Red Hat Linux that are supported by Spacewalk Server.
