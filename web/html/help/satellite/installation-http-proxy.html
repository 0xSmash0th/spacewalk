Chapter 4. Installation

4.2. Installing through an HTTP Proxy

If the target system is behind an HTTP proxy server, change the default HTTP proxy settings for the client software in /etc/sysconfig/rhn_register and /etc/sysconfig/up2date to reflect this.

After installation, if the Spacewalk Server is to remain behind an HTTP proxy, the same settings must be edited in /etc/rhn/rhn.conf. This file will not be present until after the Spacewalk Server Installation Program has completed.

Do not change these settings unless the target system is behind an HTTP proxy.
